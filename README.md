# 👻 Snapchat UI Clone

A modern and visually appealing Snapchat UI Clone built with a focus on smooth animations, clean design, and real-world app experience.

This project replicates the core UI/UX of Snapchat including camera screen, chat interface, stories section, and navigation — making it perfect for learning and showcasing frontend skills.

---

## 🚀 Features

- 📸 Camera Screen UI
- 💬 Chat Screen (Snap-style layout)
- 🔥 Stories UI (horizontal scroll)
- 👤 Profile Screen
- 🎯 Smooth Animations & Transitions
- 🌙 Dark Theme Support
- ⚡ Fast & Responsive Design
- 📱 Mobile-First UI

---

## 🛠️ Tech Stack

- Flutter 
- Dart / JavaScript
- Custom Animations
- Responsive UI Design
